Conférences professionnelles 2025

Ce projet résume les principaux enseignements des conférences sur le NLP, présentées par des experts issus de grandes entreprises et institutions de recherche.

Pour mieux comprendre ces sujets, j'ai généré des word clouds mettant en avant les termes clés, analysé les fréquences des mots et comparé les similarités entre les différentes conférences.

📌 Wordclouds

Cette section contient les nuages de mots générés à partir des conférences. Ils mettent en évidence les termes les plus utilisés pour chaque session.

Vous y trouverez :

  • 🌍 Un word cloud global combinant toutes les conférences.
  • 📊 Six word clouds individuels, un pour chaque conférence.

📌 Word Frequencies

Cette section présente des graphiques montrant les mots les plus fréquents dans chaque conférence. Cela permet de comparer quantitativement les sujets abordés.

Vous y trouverez :

  • 📊 Un top 10 des mots les plus fréquents toutes conférences confondues.
  • 📉 Six graphiques montrant les termes dominants pour chaque conférence.

📌 Similarités

Cette section explore les similarités sémantiques entre les conférences. Au lieu de compter les mots, nous utilisons des méthodes de distance cosinus pour évaluer les similitudes thématiques.

Vous y trouverez :

  • 🔥 Une heatmap des similarités entre toutes les conférences.
  • 📋 Une analyse des sessions les plus proches en termes de contenu.
